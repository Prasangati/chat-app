Hey! I am a CS student at Hunter College with an interest in web development. I am especially interested in learning more about backend technologies. I created this website to learn the inner workings of the Django framework. It is a simple chat site that utilizes a websocket connection using Django Channels and vanilla Js. Tailwind CSS is also utilized.

The choice to start one's web development journey with Django is an interesting one. Most colleges and coding bootcamps seem to focus on Javscript frameworks such as Express.js, despite the fact that Django remains an incredibly popular framework among developers. Due to my previous Python knowledge—and perhaps my desire to be a contrarian—I decided to step my foot forward with Django.

Future Plans

I want the user to be able to use their Google account to sign up and log in. I also want to utilize WebRTC API to enable video calls. Furthermore, I want to learn React to implement advanced features.
